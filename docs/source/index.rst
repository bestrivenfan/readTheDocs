Welcome to Lumache's documentation!
===================================

**切片软件部知识库** (/lu'make/) is a Python library for cooks and food lovers
that creates recipes mixing random ingredients.
It pulls data from the `Open Food Facts database <https://world.openfoodfacts.org/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

       创想云模型预览功能基本可以满足模型的常规预览需求,但就目前预览存在的一些大模型预览卡顿,模型光照视觉体验,模型场景展示单一等体验问题,以及通过对比相关竞品网站模型预览功能,发现创想云模型预览模块存在模型预览流畅性,模型操作灵活性,模型展示多样性等局限性问题。
       鉴于用户群体的多样性,为了满足各类用户对模型预览的体验需求,本次优化评审项结合各大相关竞品网站模型预览功能体验类目,取其精华功能,保留自己网站交互风格,深入3D可视化领域着手落实提升模型预览流畅性,优化模型操作灵活性,拓展模型展示可能性等模型优化预研方向。
       
       
       

.. note::

   This project is under active development.
   
   
   
   2.2 模型优化方案

1.竞品网站	
参照竞品网站的核心模型预览功能,结合ThreeJS相关API 最大限度提升实现精度和功能相似度,在此基础上,进一步优化相关功能操作体验。

2.视觉体验	
参照竞品网站的视觉体验,取其精华,收集较为优秀的模型热点动画设计,以用户视角去思考可优化方向的具体实施方案。

3.深入3D可视化领域	
借鉴 ThreeJS 官网优秀的视觉渲染手法,吸取可行的想法思路,着手实现符合创想云现有阶段的模型展示方式。

4.持续反馈	
通过不断向领导反馈优化思路,找准和把握住正确的模型优化方向,及时在不同功能的实现中调整更优方案,避免做无用功。
